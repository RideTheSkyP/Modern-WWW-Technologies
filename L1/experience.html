Experience & Projects

  • Discord-DreamBot: Music bot for discord, implemented with asynchronous discord python API. For this project I used video/audio codec ffmpeg for video-to-audio transformation in best quality. Bot is deployed and hosted on heroku server. Also created MySQL database to store users playlists, hosted on heroku too.

  • Pneumonia-Detector app: Team project. GUI application for pneumonia recognition by chest x-ray images using machine learning. The following technologies used for this app: Keras + TensorFlow for different models designs (chosen were 5 top scored models to achieve the best possible and true result, models trained and tested in Google Colab), Pillow + OpenCV for image recognition, NumPy for image-to-matrix transformations, h5py to store all matrices with labels taking up as little space as possible, Matplotlib + Seaborn to show accuracy and errors on plots and the GUI was implemented with PyQt5.

  • University website project: Team project. I created the backend. Used Django and hosted on Google Cloud Platform. Website hosted on Google App Engine and database on Google SQL database. Implemented login with Facebook. Using Trello to manage project tasks.

  • Notify project: My current project (organizer) that is being developed in order to learn how to create beautiful and comfortable (user-friendly) applications for android, based on Google Material Design. I’ve used Kivy and KivyMD frameworks.
